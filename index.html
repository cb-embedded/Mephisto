This player uses the Web Audio API to synthesize sounds based on the music data found in the 8051 firmware at addresses 0x2D00-0x3200.

Music format: Two formats are used: [Command] [Note] [Duration] or [Note] [Duration] [Command] where Command bytes are 0x68-0x72 (channel markers), Note is ASCII-like (0x31-0x51), and Duration is in time units.

The firmware contains multiple pre-stored melodies that would play on an AY-3-8910 PSG chip.
